🚗 Vehicle Management Assignment

Angular Frontend Assessment

Overview

Your task is to build a small vehicle management feature where users can view and manage their assigned vehicles.

The application currently uses mocked data. Treat the provided API service as if it was communicating with a real backend.

Required Functionality

1

Display vehicles assigned to the current user.

2

Create a form to add a new vehicle.

3

After creating a vehicle, update the UI without refreshing the page.

4

Use the provided API service as the data access layer.

Vehicle Information

  • Brand
  • Model
  • Year
  • License plate
  • Owner information

Optional Extensions

  • Edit existing vehicles
  • Delete vehicles
  • Loading and error handling
  • Additional validation
  • Improved user experience

What We Evaluate

The focus is not on visual design. We are interested in your approach to building maintainable Angular applications.

Code organization, component design, state handling, and separation of responsibilities are important parts of the assessment.
